# Style Guide

## Colour Palette

Our palette is inspired by an urban fitness aesthetic: cool neutrals, strong contrasts, and a modern, industrial feel.

- Icy — #bcc8d0
Primary background for pages and large sections.
- Common Feldspar — #848e94
Secondary text, muted labels, subtle dividers.
- Le Luxe — #5e656a
Card backgrounds, hover states, neutral UI elements.
- Dark River — #3e4447
Section headers, accent borders, subtle shadows.
- Nero — #222527
Primary border colour, containers, strong accents.
- Black Wash — #0c0e0e
Primary text colour for maximum contrast.

## Typography

### Font Family

For a workout management app, the typography should feel strong, modern, and easy to read. The font family we chose:
**_Inter_**

Fallback stack:
'Inter', system-ui, Avenir, Helvetica, Arial, sans-serif

Font sizes:

- Heading (H1): 2rem–2.5rem
- Subheadings (H2/H3): 1.25rem–1.5rem
- Body text: 1rem
- Small text / labels: 0.875rem
